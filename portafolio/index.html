Amoroma

Tipo de proyecto: Sitio web

Descripción: Este sitio fue desarrollado para darle promoción a la colonia Roma en el Distrito Federal. Se enfocó en las esperiencias que uno puede encontrar tales como: la comida, la música, áreas verdes y lugares para dormir entre otros.

Descripción técnica:

  • HTML / CSS
  • Galerías de imágenes, videos y popups
  • Se desarrolló un sistema de navegación con Javascript donde el usuario puede arrastrar la categoría de la cual qusiera conocer más y ver el contenido de la misma
  • La sección del manifiesto tiene elementos animados cuando se hace scroll
  • Social Share

Banamex Entretenimiento

Tipo de proyecto: Sitio web

Descripción: Sitio promocional de los eventos artísticos y culturales patrocinados por Banamex, en él se pueden encontrar los detalles de cada evento como un resúmen, horarios, el lugar y galerías de fotos.

Descripción técnica:

  • HTML / CSS3
  • Framework Foundation
  • Sitio Responsivo
  • Google Maps
  • Calendario
  • Galería de imágenes y videos
  • Twitter Feed
  • Social Share
  • Se alimenta de un CMS
