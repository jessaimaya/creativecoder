Amoroma

Tipo de proyecto: Sitio web

Descripción: Este sitio fue desarrollado para darle promoción a la colonia Roma en el Distrito Federal. Se enfocó en las esperiencias que uno puede encontrar tales como: la comida, la música, áreas verdes y lugares para dormir entre otros.

Descripción técnica: se utilizó HTML y CSS para el markup. El sitio tiene galerías de imágenes, videos y popups. Se desarrolló un sistema de navegación con Javascript donde el usuario puede arrastrar la categoría de la cual qusiera conocer más y ver el contenido de la misma. La sección del manifiesto tiene elementos animados cuando se hace scroll.
