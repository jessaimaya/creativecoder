Amoroma

Tipo de proyecto: Sitio web

Descripción: Este sitio fue desarrollado para darle promoción a la colonia Roma en el Distrito Federal. Se enfocó en las experiencias que uno puede encontrar tales como: la comida, la música, áreas verdes y lugares para dormir entre otros.

Descripción técnica:

  • HTML / CSS
  • Galerías de imágenes, videos y popups
  • Se desarrolló un sistema de navegación con Javascript donde el usuario puede arrastrar la categoría de la cual qusiera conocer más y ver el contenido de la misma
  • La sección del manifiesto tiene elementos animados cuando se hace scroll
  • Social Share

Banamex Entretenimiento

Tipo de proyecto: Sitio web

Descripción: Sitio promocional de los eventos artísticos y culturales patrocinados por Banamex. En cada evento se puede encontrar el horario, el lugar y un resumen.

Descripción técnica:

  • HTML / CSS3
  • Foundation Framework
  • Sitio Responsivo
  • Google Maps
  • Calendario
  • Galería de imágenes y videos
  • Twitter Feed
  • Social Share
  • Drupal CMS

Banamex Shukupá

Tipo de proyecto: Facebook App

Descripción: Aplicación en Facebook para concursar por premios para le mundial. El usuario podía crear una canción con los diferentes beats tanto de insturmentos como grabados por algunos jugadores de la selección.

Descripción técnica:

  • HTML / CSS3 / LESS
  • Bootstrap Framework
  • Animaciones CSS3 fallback JS
  • Transiciones GreenSock GSAP
  • Audio JS
  • Sistema de Ranking
  • Galería de concursantes
  • Facebook App
  • Social Share
  • Acciones como reproducir, guarar, reproducir todo, detener, siguiente y anterior
  • Almacenamiento de canciones (JSON)
